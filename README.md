# Ritik Garg's Portfolio

A modern, elegant, and professional personal portfolio website for me as Software Developer, built with **React**, **TypeScript**, and **Tailwind CSS**. This portfolio features smooth animations, a dark/light mode toggle, and an integrated AI chatbot powered by Google Gemini API.

Follow these steps to set up and run the project locally:

1. **Clone the repository**
   ```sh
   git clone https://github.com/yourusername/developer-portfolio.git
   cd developer-portfolio
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

